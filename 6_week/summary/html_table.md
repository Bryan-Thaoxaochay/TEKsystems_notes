# HTML Table
```
<table>
  
  <tr>
    <th></th>
    <th></th>
    <th></th>
  </tr>
  
  <tr>
    <td></td>
    <td></td>
    <td></td>
  </tr>
  
  <tr>
    <td></td>
    <td></td>
    <td></td>
  </tr>

</table>
```
- table: indicates browser that it is a table
- tr: row in the table
- th: column name
- td: cell to contain the data
